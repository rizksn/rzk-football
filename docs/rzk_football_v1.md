
# 🧠 Anubis Draft Cockpit – Product Vision

This is not just a mock draft simulator. This is the **ultimate command center** for serious fantasy football drafters — a fully customizable, AI-powered, data-rich cockpit designed to help users dominate every draft format and scenario.

---

## 🚀 Core Philosophy

### 🧭 All-in-One Draft Control Panel
A centralized tool for mock drafting, studying, strategizing, note-taking, and preparation — covering every league type:
- Redraft
- Dynasty
- Rookie drafts
- 2QB / Superflex
- Keeper leagues

### 🧠 Year-Round Utility, Not Just Draft Day
Even when not drafting, users can use the cockpit as a **war room**:
- Queue players
- Write notes
- Track risers/fallers
- Rehearse real-world trade and draft scenarios

### 🧱 Customization at Every Level
Users build their draft room like **fantasy LEGO blocks**:
- Show/hide panels
- Reorder views
- Choose data overlays
- Save layouts per league

### 🦾 Tuned AI Draft Engine
The CPU engine is **user-configurable**:
- Positional strategy (RB-heavy, Zero WR, etc.)
- Risk tolerance and randomness
- Team need prioritization
- Aggression sliders and variance controls

---

## 🧩 Long-Term Goals

### ✅ Trusted Draft Hub for the Fantasy Community
This becomes the **default mock tool**. When August hits, users say:
> "Load up the cockpit."

### 📊 Deep Data Integrations
- ADP from multiple sources
- Optional access to premium APIs like PFF, FantasyPros, or Rotowire
- Injury and depth chart overlays
- Custom scoring calculations

### 🛠️ Custom League Simulations
Users save league profiles with:
- Roster structure
- Scoring settings
- Number of rounds
- Keepers (if applicable)
- Auto-loaded into the draft simulator

### 🎓 Continuous Education
The platform teaches mock drafting like a skill:
- Scenario practice
- Strategy testing
- Retrospective analysis of picks
- “Coach Mode” explains AI logic in real-time

---

## 📐 Tech Stack & Infrastructure

- **Frontend:** Next.js + TypeScript + Tailwind
- **Backend:** FastAPI + Python + GPT integration
- **Draft Engine:** Modular scoring + GPT decider layer
- **Data Fetching:** ADP scrapers via Playwright, auto-refresh nightly
- **Database:** Postgres (via Supabase or SQLAlchemy)
- **Auth:** Email or OAuth login
- **Monetization:** Tiered access + format unlocks
